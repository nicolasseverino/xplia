## ğŸ’¡ How to Use XPLIA

### Quick Start (30 seconds)

```python
from xplia import create_explainer
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris

# Load data
X, y = load_iris(return_X_y=True)

# Train model
model = RandomForestClassifier()
model.fit(X, y)

# Create explainer (auto-detects best method)
explainer = create_explainer(model, method='shap')

# Generate explanations
explanation = explainer.explain(X[:5])
print(explanation.feature_importance)
```

### For Compliance (GDPR, EU AI Act)

```python
from xplia.compliance import GDPRCompliance, AIActCompliance

# GDPR
gdpr = GDPRCompliance(model, model_metadata={'name': 'Credit Model'})
dpia_report = gdpr.generate_dpia()
dpia_report.export('gdpr_report.pdf')

# EU AI Act
ai_act = AIActCompliance(model, usage_intent='credit_scoring')
risk = ai_act.assess_risk_category()  # Returns 'HIGH' for credit
compliance_report = ai_act.generate_compliance_report()
```
